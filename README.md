## MANEJO DE EXCEPCIONES 
Esta actividad tiene como objetivo aplicar los conceptos fundamentales del manejo de excepciones en Java, comprendiendo cómo prevenir fallos inesperados durante la ejecución de un programa y cómo controlarlos adecuadamente.
Se trabajó con:
-Excepciones estándar del lenguaje.
-Creación de excepciones personalizadas.
-Uso de try, catch, finally.
-Uso de throw y throws.
-Propagación de excepciones entre métodos.
